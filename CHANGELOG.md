# Changelog

## 0.1.0

- Initial native macOS menu bar app
- Connects to paired Sony WH-1000XM4 headsets over classic Bluetooth RFCOMM
- Noise Cancelling, Ambient Sound, and Sound Control Off controls
- Ambient level and Focus on Voice controls
- Battery status from macOS power reports
- macOS output volume slider
- Debug protocol logging via `BOREA_DEBUG=1`
